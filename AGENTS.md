# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目简介

`my-knowledge` — 个人知识库。用户需要记录内容时，使用 `.md` 文件写入此仓库。

## 使用方式

- 用户说"记录一下"、"记下来"、"保存"等意图时，将内容整理为 `.md` 文件写入项目根目录或合适的子目录。
- 文件名使用有意义的英文 slug 命名（如 `flutter-state-management.md`）。
- 内容使用 Markdown 格式，结构清晰，便于后续检索和阅读。
- 同一主题的内容追加到已有文件，不重复创建。
