# iOS 开发工具

## UDID 查询

**蒲公英提供的在线工具**，iOS 手机在 Safari 中打开即可快速查看本地 UDID，无需通过 Xcode 或 Finder。

- 地址：<https://www.pgyer.com/tools/udid>
- 用途：获取 iPhone/iPad 的 UDID，用于内测设备注册、证书配置等
- 使用方式：Safari 打开链接 → 按提示安装描述文件 → 获取 UDID

### 操作步骤

1. iPhone 用 Safari 浏览器打开 <https://www.pgyer.com/tools/udid>
2. 点击"获取 UDID"按钮
3. 允许安装描述文件（设置 → 通用 → VPN 与设备管理 → 安装）
4. 安装完成后自动跳转回网页，页面显示 UDID
5. 可复制或发送至邮箱
